# WhatsApp Chatbot API

## Description
This is a simple WhatsApp chatbot backend simulation built using Spring Boot. It receives messages via a REST API and responds with predefined replies.

## API Endpoint
POST /webhook

## Sample Request
{
  "message": "hi"
}

## Sample Response
{
  "reply": "Hello"
}

## Features
- REST API using Spring Boot
- DTO-based clean architecture
- Logging using SLF4J
- Input validation
- Predefined chatbot responses

## How to Run
1. Clone the repository
2. Open in IntelliJ or any IDE
3. Run the Spring Boot application
4. Test using Postman
